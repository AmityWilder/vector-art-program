# equal
perform a component-wise equal-to comparison of two vectors

## Parameters
- `x`
  Specifies the first vector to be used in the comparison operation.

## Description
[`Gl::equal`] returns a boolean vector in which each element *i* is
  computed as `x`[*i*] == `y`[*i*].
