# sqrt
return the square root of the parameter

## Parameters
- `x`
  Specify the value of which to take the square root.

## Description
[`Gl::sqrt`] returns the square root of $x$, i.e. the value $\sqrt { x
  }$. The result is undefined if $x < 0$.
