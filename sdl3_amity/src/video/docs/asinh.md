# asinh
return the arc hyperbolic sine of the parameter

## Parameters
- `x`
  Specify the value whose arc hyperbolic sine to return.

## Description
[`Gl::asinh`] returns the arc hyperbolic sine of $None$; the inverse
  of $$ None $$ *x*[`Gl::sinh`].
