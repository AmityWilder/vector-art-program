# isnan
determine whether the parameter is a number

## Parameters
- `x`
  Specifies the value to test for NaN.

## Description
For each element *i* of the result, [`Gl::isnan`] returns [`true`] if
  `x`[*i*] is positive or negative floating point NaN (Not a Number) and
  false otherwise.
