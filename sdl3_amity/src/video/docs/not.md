# not
logically invert a boolean vector

## Parameters
- `x`
  Specifies the vector to be inverted.

## Description
[`Gl::not`] logically inverts the boolean vector `x`. It returns a new
  boolean vector for which each element *i* is computed as ```c ! ``` .
  `x`[`i`]
